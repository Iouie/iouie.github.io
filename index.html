Work

KR33-8

KR33-8 is a team created for our final Capstone Project. It involves the idea of teaching people about terraforming. There are 3 educational minigames that explain the processes of terraforming and how to create the perfect planet.

The Heist

The Heist is a recreated 2D multi-directional shooter game from the classic game Asteroids. The player must avoid and shoot down as many enemy cars as possible and in order to rack up points. Developed with Unity and C#.

Demon Visualizer

The Demon Visualizer displays visuals based on a music files frequency and has many other options the users can select. Developed using Canvas API, Web Audio API and Flexbox.

Dogfighter

Dogfighter is a 2D shoot 'em up similar to Atari's Galaga. The player must destroy enemy jets before they reach the bottom of the screen. Developed using Swift and Apple's SpriteKit framework.
